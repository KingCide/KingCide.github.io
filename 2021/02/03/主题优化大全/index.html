文章中添加图片

（只有2、3、9、10步有用）

  1. 安装图片插件npm install hexo-asset-image --save

  2. 在_config.yml配置文件中，修改为post_asset_folder: true。这样之后每次新建文章，都会出现与.md同名的文件夹用来保存所需内容。

  3. 出现问题：在Typora中，照片若要正常显示需要相对路径如![p](./文件夹名字/p.jpg)，然而此时无论什么路径网页中都无法显示。

  4. 删除插件：npm remove hexo-asset-image

  5. 安装插件：cnpm install https://github.com/CodeFalling/hexo-asset-image

  6. 此时若使用路径![p](p.jpg)可以在网页中正常显示，然而Typora中没有预览。

  7. 继续删除插件npm uninstall hexo-asset-image --save不过好像没有一点作用。此时，若将照片保存在.md同名文件夹，可以用路径![p](p.jpg)在详情页显示而主页不显示；若将照片保存在./source/image，则可以用路径![q](/image/q.jpg)在主页和详情页中显示；若将照片保存在./source/image，用路径![q](./image/q.jpg)在主页中显示而详情页中不显示。

  8. 无语。

  9. 病急乱投医：cnpm install https://github.com/7ym0n/hexo-asset-image --save

  10. 这个插件一来就好多了，至少没有主页和详情页显示不同的问题。且下面三种方式都可以显示图片：

  11. 现在突然发现不能部署，部署报错Spawn failed及解决方案 。主要原理就是要回退到某个版本。
