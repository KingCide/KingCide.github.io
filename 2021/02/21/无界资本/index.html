无界资本

发表于 2021-02-21 更新于 2021-03-07 分类于 学习印记 阅读次数：

摘要

本书由沈亦文所著，出版于2015年。

本书梳理了中国金融领域最近几年发生的一系列大事件，用金融破局串联起了这些事件背后的关系。全书分为三篇，第一篇概述了在资本狂飙时代，产业资本与金融资本的种种乱象，提出问题，指出中国金融的潜在危机。第二篇则高呼狼来了，讲述互联网思维、互联网跨界与互联网金融对于传统金融的冲击，以及这一切将带来的机会与挑战。第三篇又回到传统金融与产业，讲述在互联网汹涌来袭时代金融业的未来发展趋势与重生之路。

它告诫我们，新生事物出现并颠覆现有的商业模式和盈利基础，而我们不得不以一种全新的视角审视我们的世界。

流动性狂飙时代的资本困兽

流动性盛宴下的产业迷思

  1. 4万亿狂欢后的钢贸企业之殇：央行增发货币，银行争相放贷 -> 钢贸企业变成了融资平台，贷款被投入大量未经银行风险评估的其他领域 -> 直至钢材价格猛跌等问题出现，使资金链越来越紧 -> 银行抽贷 -> 出现重复质押，联保互保连锁崩塌。究其原因，可能是4万亿宏观调控、可能是钢贸商未能坚守本业稳步发展、可能是银行一心赚快钱、可能是联保互保的信贷模式……

    在经济发展的快车道里，所有人都在以各种方式掠夺财富。在easy money面前，人们想的是如何一夜暴富，而不是想着如何诚信经营，打造一个百年老店。
