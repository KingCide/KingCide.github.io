计算机通信网络-4-介质控制子层

发表于 2021-05-07 更新于 2021-05-19 分类于 学习印记 阅读次数：

MAC层概述

广播式信道

  • 光纤+光柱
  • 双绞线+集线器
  • 同轴电缆+T型头
  • 环网

遇到的问题

  • 两个以上站点使用共享信道
  • 不能有多个站点同时发送
  • 接收站识别送给自己的数据

问题解决目标

  • 低成本——局部范围网络、分布式、无中心
  • 能降低冲突或无冲突
  • 尽量提高信道利用率

共享信道技术

  • 分路复用技术 xDM
  • 分路多路访问（多址）技术 xDMA
  • 多路访问技术 xMA

竞争式访问协议

ALOHA协议

  1. 随时发送数据，等待接收方确认=对发送没有控制，用停等协议解决差错。

  2. 冲突现象严重。

  3. 时槽式ALOHA

    • 将信道使用时间分成离散的时槽。
    • 冲突要么完全重叠，要么完全分开。

CSMA协议

  1. 综述
    • 载波侦听多路访问（Carrier Sense Multiple Access）
    • 发送前侦听线路，空闲则立即发送，若信道忙则监听直到信道空闲
    • CSMA冲突窗口：$ T_p=\frac{d}{2/3c} $
  2. 1坚持CSMA：信道由忙转闲时，以概率1发送数据
  3. P坚持CSMA：信道由忙转闲时，以P概率发送，(1-P)概率随机延时后再尝试发送
  4. 0坚持CSMA：信道由忙转闲时，以0概率发送，都需要随机延时后再尝试发送

CSMA/CD协议

  1. 载波侦听多路访问/冲突检测
  2. 冲突检测时间
  3. 随机延时算法：截断二进制指数回退算法
    • 最小时间片是最大冲突检测时间
    • 根据尝试重传的次数变化延时的范围（二进制指数增加，$2^{10} $后不再增加）

CSMA/CA协议

  1. 无线局域网技术
  2. 冲突域不固定
  3. 通信问题
    • 隐藏站点：RTS（Require to Send）+CTS（Clear to Send）
    • 暴露站点

非竞争式协议

无冲突协议

  1. 预定协议——信道申请
  2. 自定方式——二进制倒计数法
  3. 用特定信息指挥发送——令牌环

以太网

采用了CSMA/CD技术的局域网

共享式以太网

  1. 物理层接口：
    • BNC——同轴电缆接口
    • RJ45——双绞线接口
    • SC——光纤接口
  2. 主要采用曼彻斯特编码
  3. 媒体访问技术——CSMA/CD
  4. 最小帧长64字节，最大帧长1518字节，最多连续冲突次数16次
  5. 以太网编制：MAC地址6字节——单播组播广播

网桥及交换式以太网

  1. 在隔离不同局域网之间的流量的基础上实现联通
  2. 核心技术
    • 存储转发
    • 根据帧内目的地址转发
    • 互联不同LAN协议转换技术
  3. 常用技术
    • 透明网桥
    • 源路由网桥——源端指明路由路径
    • 远程网桥
  4. 交换机——多端口桥
  5. 交换式以太网

虚拟以太网

  1. 软件上划分VLAN
  2. 按端口、MAC地址、用户分
  3. VLAN协议添加头部，特殊的Trunk接口

其他局域网

  1. 无线局域网（WLAN WIFI）IEEE802.11系列协议
  2. 固定宽带WiMax
  3. 蓝牙
