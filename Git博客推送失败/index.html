Git博客推送失败

发表于 2021-05-19 更新于 2026-02-19 分类于 学习印记 阅读次数：
解决 Hexo 博客 Git 推送失败的排查过程和解决方案。

遇到的问题

  • 在将文章发布到博客时，总是报错“Spawn failed”，具体报错和网上其他人相同。

解决过程

主要参考资料

  1. 这里列举了各种解决方式https://blog.zhheo.com/p/128998ac.html
  2. https://blog.csdn.net/HTL2018/article/details/106876940
  3. https://1187100546.github.io/2019/11/24/spawn-failed/
  4. 查看commit ID https://www.jianshu.com/p/ba15bf62bcc7
  5. 新建SSH和init仓库https://www.cnblogs.com/xsilence/p/6001938.html

个人尝试过程

  1. 删除.deploy_git，重试，无效

  2. 尝试回退commit ID，无效，fatal: Not a git repository

  3. 尝试连接github，发现连接不上，到github发现ssh没了

  4. 于是重置ssh，更改host的github的IP

  5. 最后成功以较快的速度上传
